# 🚀 SwiftLink

**The easiest way to get paid on Celo. Built for MiniPay.**

SwiftLink allows anyone to generate a unique payment link (e.g., `swiftlink/pay/john`) and receive cUSD/CELO payments directly to their wallet in seconds. Designed for the Celo ecosystem and optimized for the MiniPay mobile experience.

---

## ✨ Features

- **Personalized Payment Pages**: Create your own permanent link to share on social media.
- **AI-Powered Invoicing**: Generate professional descriptions with a built-in AI assistant ✨.
- **QR Code Suite**: Instantly share your payment link via high-res QR codes, optimized for in-person payments with one-click PNG export.
- **Cinematic Dashboard**: A premium, glassmorphic analytics hub to track volume, unique payers, and real-time transaction history.
- **On-chain Receipts**: Detailed, animated transaction modals with direct Celoscan integration.
- **Mobile-First Navigation**: Dedicated bottom navigation bar optimized for MiniPay and mobile browsers.
- **Data Portability**: Export your entire transaction history to CSV with a single click.
- **Verified Identity**: Simulated verified badges and social proof for every creator.
- **Dynamic Social Previews**: Premium OG cards for every payment link, perfect for sharing on X and Farcaster.
- **Zero Fees**: Direct peer-to-peer payments on the Celo network with near-zero gas costs.

---

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion, Radix UI
- **Web3**: Viem, Wagmi, RainbowKit, Celo SDK
- **Smart Contracts**: Solidity, Hardhat, OpenZeppelin
- **Network**: Celo Mainnet (Production)
- **Design**: Cinematic Dark-Mode with Custom Mesh Gradients & Glassmorphism
